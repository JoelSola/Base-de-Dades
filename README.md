# 📚 BASES DE DATOS

Aquí muestro las actividades que hice durante el curso de bases de datos. ¡Explora los ejercicios y configuraciones paso a paso!

## 📋 Resumen de Actividades

| Actividad | Descripción | Enlace |
|-----------|-------------|--------|
| **1** | Instalación y configuración inicial de MySQL, Percona Server y Redis. | [Ver detalles](https://github.com/JoelSola/Base-de-Dades/tree/main/Activitat%201) |
| **2** | Configuración de logs en MySQL (general, slow query, binary). | [Ver detalles](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%202/Logs.md) |
| **3** | Motores de almacenamiento en MySQL (MyRocks, CSV, InnoDB, etc.). | [Ver detalles](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%203/storageengine.md) |
| **4** | Replicación en MySQL (master-slave). | [Ver detalles](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%204/Replicaci%C3%B3.md) |

## 🛠️ Actividad 1
En esta actividad realicé la documentación de la instalación y configuración inicial de los sistemas de bases de datos siguientes:
- **[MySQL](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%201/MySQL.md)** 🐬
- **[PerconaServer](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%201/PerconaServer.md)** 🔧
- **[Redis](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%201/Redis.md)** ⚡

## 📝 Actividad 2
En la siguiente actividad realicé ejercicios relacionados con la documentación de configuración de logs en MySQL, incluyendo activación de logs general, slow query y binary mediante archivos de configuración (`logs.cnf` y `my.cnf`), verificación de estados, desactivación de logs, cambio de destino a tablas, carga de la base de datos Sakila, conteo de sentencias en logs, ejecución de queries lentas, gestión de binary logs (reset, flush, purge), y uso de `mysqlbinlog`:
- **[Ejercicios Logs](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%202/Logs.md)** 📊

## 🔧 Actividad 3
En la siguiente actividad realicé ejercicios centrados en la documentación sobre motores de almacenamiento en MySQL, incluyendo la identificación de motores activos, cambio del motor por defecto (ej. a MyISAM), instalación y activación de MyRocks, uso del motor CSV, configuración de InnoDB (como desactivar `innodb_file_per_table`), importación de la base de datos Sakila, creación de tablespaces personalizados, y verificación de LSN y checkpoints:
- **[Storage Engines](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%203/storageengine.md)** 🗂️

## 🔄 Actividad 4
En esta actividad realicé ejercicios relacionados con la documentación del proceso de replicación en MySQL, incluyendo la configuración de un servidor maestro (master) y esclavo (slave), con pasos detallados como verificación de `server-id`, flush de logs, creación de backups, edición de archivos de configuración (`my.cnf`), y comandos para iniciar la replicación:
- **[Replicación](https://github.com/JoelSola/Base-de-Dades/blob/main/Activitat%204/Replicaci%C3%B3.md)** 🔄
